# Лабораторная работа 2 - WPF приложение (Вариант 8)

## Описание
WPF приложение для вычисления математических выражений по 4 формулам.

## Функциональность
- Выбор одной из 4 формул
- Ввод параметров a, b, c
- Вычисление результата
- Обработка ошибок

## Технологии
- WPF
- C#
- XAML
